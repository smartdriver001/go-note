## 命令
1. go build 编译并安装自身包和依赖包
1. go install 安装自身包和依赖包
1. [gofmt](https://golang.org/cmd/gofmt/)格式化
1. [godoc](golang.org/cmd/godoc/) 生成代码文档, godoc -http=:6060

## Go 命名规范
1. 文件名小写
1. 函数,方法,变量名采用驼峰命名法
1. 接口以"er"结束
